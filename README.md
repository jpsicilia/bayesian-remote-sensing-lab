# Bayesian Remote Sensing Lab

> "A sensor does not measure reality. It measures reality plus noise.
> Understanding that noise is the foundation of all probabilistic inference."

## What this is

A progressive, hands-on learning repository for Bayesian statistics applied to Earth Observation.

Built around real Sentinel-2 data from Parana, Brazil.
